概述

Velocity 是一个简单易用、高性能、功能丰富的轻量级JS动画库。它能和 jQuery 完美协作，并和$.animate()有相同的 API， 但它不依赖 jQuery，可单独使用。 Velocity 不仅包含了 $.animate() 的全部功能， 还拥有：颜色动画、转换动画(transforms)、循环、 缓动、SVG 动画、和 滚动动画 等特色功能。

它比 $.animate() 更快更流畅，性能甚至高于 CSS3 animation， 是 jQuery 和 CSS3 transition 的最佳组合，它支持所有现代浏览器，最低可兼容到 IE8 和 Android 2.3。
velocity官网: http://velocityjs.org/
中文文档: http://www.mrfront.com/docs/velocity.js/
github: https://github.com/julianshapiro/velocity/

动画定义

预定义动画

velocity已经帮我们预定义好了大批量的动画效果，可以通过官网查询。我们可以使用这些已经定义好的动画，来快速完成我们要的效果。

自定义动画

如果velocity定义的动画还无法实现我们要的动画效果，开发人员也可以自己定义一个动画。自定义动画需要用 RegisterEffect() 或 RegisterUI() 函数来实现。这两个函数实现的效果是一样，在开发中想用哪种就用哪种。
RegisterEffect(‘动画名称’,{动画参数})
