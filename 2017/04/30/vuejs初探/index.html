vue概念

Vue.js（读音 /vjuː/，类似于 view） 是一套构建用户界面的渐进式框架。与其他重量级框架不同的是，Vue 采用自底向上增量开发的设计。Vue 的核心库只关注视图层，它不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与单文件组件和 Vue 生态系统支持的库结合使用时，Vue 也完全能够为复杂的单页应用程序提供驱动。

vue中的两个核心点

响应的数据绑定

当数据发生改变–> 自动更新视图
利用Object.definedProperty中的setter/getter代理数据，监控对数据的操作

组合的视图组件

ui页面映射为组件树
划分组件可维护，可重用，可测试

虚拟DOM

javascript运行js的速度是很快的，大量的操作DOM就会很慢，时常在更新数据后重新渲染页面，这样造成在没有改变数据的地方也重新渲染了DOM节点，这样就造成了很大程度上的资源浪费。
概念
利用在内存中生成与真实DOM与之对应的数据结构，这个在内存中生成的结构称之为虚拟DOM。
当数据发生变化时，能够智能的计算出重新渲染组件的最小代价并应用到DOM操作上。

基础语法

vue实例

每一个应用都是通过Vue这个构造函数创建根实例(root instance)启动new Vue(选项对象)
需要传入选项对象，对象包含挂载元素，数据，模板，方法等。

el: 挂载元素选择器          String | HtmlElement
data: 代理数据              Object | function
methods: 定义方法           Object

v-on

v-on:事件名=函数名
当我们往methods里传入事件函数的时候，需要在需要绑定事件的元素上添加 v-on:事件名=函数名，以便完成事件绑定。

示例代码 
<body>
<script src="vue.js"></script>
<div id="demo">
    <span v-on:click="clickHandle">{{message}}</span>
</div>
    <script>
        let obj = {
            message: 'hello,vue!'
        }
        new Vue({
            el: '#demo',
            data: obj,          //传入数据对象，也可以直接写数据message: 'hello,vue!'
            methods: {
                clickHandle(){
                    alert('click');
                }
            }
        })
    </script>
</body>

vue代理data数据

每个Vue实例都会代理其data对象里所有的属性，这些被代理的属性是响应的。新添加的属性不具备响应功能，改变后不会更新视图。

vue实例自身属性和方法

暴露自身的属性和方法，以$开头，例如：$el $data…..

示例代码 
<script>
    let obj = {
        message: 'hello,vue!'
    }
    var vm = new Vue({
        el: '#demo',
        data: obj,          //传入数据对象，也可以直接写数据message: 'hello,vue!'
        methods: {
            clickHandle(){
                alert('click');
            }
        }
    });
    close.log(vm.$data)     //通过这种方法可以拿到数据
</script>

声明式渲染

Vue.js 的核心是一个允许采用简洁的模板语法来声明式的将数据渲染进 DOM：

声明式

只需要声明在哪里(where)做什么(what),而无须关心如何实现(how)

命令式

需要一具体代码表达在哪里(where)做什么(what),如何实现(how)
例子：求数组中每一项的倍数

命令式 
使用for循环拿出每一项，然后求职完成后，在放入另一数组中
<script>
        var arr = [1,2,3,4,5];
        var newArr = [];
        for(var i=0; i<arr.length; i++){
            newArr.push(arr[i]*2);
        }
        alert(newArr);
    </script>
声明式 
使用map方法，关注如何求职
var arr = [1,2,3,4,5];
    var newArr = arr.map(function(item){
        return item*2;
    })
    alert(newArr);

vue声明式渲染

初始化根示例，vue自动将数据绑定在DOM模板上

示例代码 
<div id="app-2">
  <span v-bind:title="message">
    鼠标悬停几秒钟查看此处动态绑定的提示信息！
  </span>
</div>

var app2 = new Vue({
  el: '#app-2',
  data: {
    message: '页面加载于 ' + new Date()
  }
})

指令（vue中的内置指令）

什么是指令？
是一种特殊的自定义行间属性
指令的职责就是当其表达式的值发生改变时相应地将某些行为应用到DOM上
在Vue中，指令以v-开头

v-bind

动态的绑定数据，简写为 ：

示例代码 
<body>
<script src="vue.js"></script>
<div id="demo">
    <span :cutomId='id'>{{message}}</span>   //绑定动态数据
</div>
    <script>
        let obj = {
            message: 'hello,vue!',
            id: 'user'

        }
        new Vue({
            el: '#demo',
            data: obj
        })
    </script>
</body>

v-on

绑定事件监听器，简写为 @

示例代码 
<body>
<script src="vue.js"></script>
<div id="demo">
    <span @click="clickHandle">{{message}}</span>  //绑定事件
</div>
    <script>
        let obj = {
            message: 'hello,vue!'
        }
        new Vue({
            el: '#demo',
            data: obj,
            methods: {
                clickHandle(){
                    alert('click');
                }
            }
        })
    </script>
</body>

v-text

更新数据，会覆盖已有结构

v-html

可以解析数据中的html结构

v-show

更具值的真假，切换元素的display属性

v-if

根据值的真假，切换元素会被销毁，重建

v-else-if

多条件判断，为真则渲染

v-else

条件都不符合渲染

v-for

基于源数据多次渲染元素或模板块

v-model

在表单空间元素上创建双向数据绑定

v-pr

调过元素和子元素的编译过程

v-once

值渲染一次，随后数据更新不重新渲染

v-cloak

隐藏为编译的Mustache语法，css中设置[v-cloak]{display:none}

模板

vue中有三种模板

html模板

基于DOM的模板。模板都是可解析的有效的HTML

插值
文本： 使用”Mustache”语法（双大括号）
作用：替换实例上的属性值，当值改变时，插值内容处会自动更新
示例代码 
<div id="demo">
    <span>{{message}}</span>
</div>
原生的html： 双大括号输出的是文本，不会解析html。
示例代码 
<body>
<script src="vue.js"></script>
<div id="demo">
    <span>{{html}}</span>    //html输出出来的是"<div>hello,daxia</div>"
</div>
    <script>
        let obj = {
            html:"<div>hello,daxia</div>"
        }
        new Vue({
            el: '#demo',
            data: obj
        })
    </script>
</body>

可用指令 v-html:html来解决原生html不会解析的问题

示例代码 
<body>
<script src="vue.js"></script>
<div id="demo">
    <div v-html="html"></div>
</div>
    <script>
        let obj = {
            html:"<div>hello,daxia</div>"
        }
        new Vue({
            el: '#demo',
            data: obj
        })
    </script>
</body>
属性： 使用v-bind进行绑定，可以响应变化
使用javascript表达式： 写简单的表达式
模板里可以写加减乘除运算，也可以写三目运算，但不推挤写在这里面，最好只把它用来单纯的渲染数据
示例代码 
<div id="demo">
    {{true ? "yes" : "no"}}     //模板里可以写加减乘除运算，也可以写三目运算，但不推挤写在这里面，最好只把它用来单纯的渲染数据
    <div v-html="html"></div>
</div>

字符串模板

template字符串

template选项对象的属性

1. 模板将会 替换挂载的元素，挂载元素的内容都将被忽略
2. 根节点只有一个
示例代码 
<body>
<script src="vue.js"></script>
<div id="demo">
    <span>str</span>
</div>
    <script>
        let obj = {
            abc: 1
        }
        var str="<div>hello,<span>daxia</span></div>"   //根节点只有一个，不能写兄弟节点，但是可以在内部写子孙节点
        new Vue({
            el: '#demo',
            data: obj,
            template: str       //通过template来挂载str字符串。并且会替换掉#demo里面的所有内容
        })
    </script>
</body>

关于字符串拼接：
在ECMAScript3中，字符串只用单引号或双引号来拼接，这种拼接方式有个缺点,就是不能换行。
在ECMAScript6中提过一个新方法 `` ,通过``来拼接字符串可以随意换行。
3. 将html结构写在一对script标签中，设置type=”x-template”，并且加上一个id，然后在vue的配置项里用template挂载这个id。
这种方法不能跨页面来调用，只能在本页面有效。
示例代码 
<body>
<script src="vue.js"></script>
<div id="demo">
    <span>str</span>
</div>
<script type="x-template" id="temp">
    <div>
        hello,
        <span>daxia</span>
    </div>
</script>
    <script>
        let obj = {
            abc: 1
        }
        new Vue({
            el: '#demo',
            data: obj,
            template: "#temp"
        })
    </script>
</body>

模板-render函数

render函数

通过render函数来手写代码, render (cleateElement) {}
render 选项对象的属性
cleateElement(标签名,[数据对象,子元素]); 子元素为文本或数组
render函数更接近vue编译器，很强大，一般写组件的时候会用到。

数据对象属性

class:{} //绑定class，和v-bind:class 一样的API
style:{} //绑定样式，和v-bind:style 一样的API
attrs:{} //添加行间属性
domProps:{} //DOM元素属性
on:{} //绑定事件

nativeOn:{} //监听原生事件
directives:{} //自定义指令
scopedSlots:{} //slot作用域
slot:{} //定义slot名称
key:”key” //给元素添加唯一表示
ref:”ref” //引用信息

示例代码 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>vue实例</title>
    <style>
        .bg {
            background: green; color: #fff;
        }
    </style>
</head>
<body>
<script src="vue.js"></script>
<div id="demo"></div>
    <script>
        let obj = {
            abc: 1
        }
        new Vue({
            el: '#demo',
            data: obj,
            render(createElement){
                return createElement(
                        "ul",
                        {
                            class: {
                                bg: true        //添加class
                            },
                            style: {
                                fontSize: '20px'    //添加行间style
                            },
                            attrs: {
                                user: 'daxia'       //添加自定义属性
                            },
                            domProps: {
                                innerHTML: '<li>我是html</li>'
                                //给ul里面添加内容，会覆盖掉下面数组中的内容，因为它的优先级会高一些
                            },
                            on: {
                                click: function(){
                                    alert(123);         //添加事件
                                }
                            }
                        },
                        [
                            createElement("li",4),
                            createElement("li",2),
                            createElement("li",3)
                        ]
                    );
            }
        })
    </script>
</body>
</html>
vue中的class绑定
示例代码 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>class绑定</title>
    <style>
        .red { color: red; }
        .fontSize { font-size: 20px; }
    </style>
</head>
<body>
<script src="vue.js"></script>
<div id="demo">
    <span class="fontSize" :class='{red:addClass}'>hello,daxia</span>
    1. 原生书写的class和vue中绑定的class可以共存
    2. 通过v-bind来绑定，v-bind会解析字符串中的代码，所以可以做判断
    3. 判断，如果addClass为true，就添加.red这个class
    4. vue中一起以数据为先
</div>
    <script>
        let obj = {
            addClass: true
        }
        new Vue({
            el: '#demo',
            data: obj
