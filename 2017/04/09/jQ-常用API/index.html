jQ初探-常用API

发表于 2017-04-09 | 分类于 笔记类

JQ下的常用API，个人笔记，方便查询。

选择元素：

选择ID： 

$('#div1').css('background','red');

选择class：

$('.box').css('background','red');

选择标签：

$('div').css('background','red');
$('li:first').css('background','red');  第一个  first
$('li:last').css('background','red');  最后一个  last
$('li:eq(2)').css('background','red'); //根据下标选择第几个的元素  eq(n)
$('li:even').css('background','red');  //选择奇数行 even
$('li:odd').css('background','red');  //选择偶数行  odd
PS: 选择第几个的时候必须加“ ： ”。
$('li.box').css('background','red');

$(function):

相当于JS下的window.onload = function(){},但是比它的性能好

<window,onload表示全部加载完再执行，而$(function(){})则是加载完DOM就执行
全写形式=$(document).ready(function(){})
JS下DOMcontentloaded具有相同的意思
兼容形式 

if (document.addEventListener){
    document.addEventListener("DOMContentLoaded", init, false);
}

方法函数化：

$(function(){
    $('#div1').click(function(){
        alert($(this).html());
    })
})

$()下的常用方法：

attr():

可以进行属性的取值和赋值操作

<script src="jquery-1.10.1.min.js"></script>
 <script>
    $(function(){
        $('div').attr('title','456');  //可以经行属性操作赋值
        $('div').attr('class','box');
                alert($('div').attr('title'));  //也可以经行属性取值
    })
</script>
</head>
<body>
    <div title="123" ></div>
</body>

filter():

对元素进行筛选过滤，针对当前元素
筛选：

$('li').filter('.box').css('background','green');

筛选自定义属性：

$('li').filter('[title=hello]').css('background','green'); //可以筛选自定义属性

not():

filter()的反义词，针对当前元素
示例：

<script src="jquery-1.10.1.min.js"></script>
 <script>
     $(function(){
        $('div').css('background','red');  //所有div背景变红
        $('div').filter('.box').css('background','green');  //当前元素带有.box的div变绿
        $('div').not('.box').css('background','blue');  //当前元素不带有.box的变蓝
     })
 </script>
</head>
<body>
    <div class="box">div1</div>
    <div>div2</div>
</body>

元素是否包含，当前元素里面的东西

$('div').has('.box').css('background','blue'); //元素里面包含.box的变蓝
})
</script>
</head>
<body>
    <div>div1<span class="box">span</span></div>
    <div>div2</div>
</body>

上一个兄弟节点

下一个兄弟节点

siblings():

获取元素所有兄弟节点，但不包括它自身，可以传入一个参数，传入的参数是你要筛选的兄弟节点

        $('span').siblings('h2').css('background','red'); //span的兄弟节点h2背景变红
    });                                                        
</script>
</head>
<body class="box">

nextAll():

获取元素下面的所有兄弟节点，但不包括它自身和它上面的兄弟节点，
可以传入一个参数，传入的参数是你要筛选的兄弟节点

prevAll():

获取元素上面的所有兄弟节点，但不包括它自身和它下面的兄弟节点，
可以传入一个参数，传入的参数是你要筛选的兄弟节点

nextUntil():

获取元素下面的所有兄弟节点，但不包括它自身和它上面的兄弟节点，
可以传入一个参数，传入的参数是你要截止的兄弟节点

prevUntil():

获取元素上面的所有兄弟节点，但不包括它自身和它下面的兄弟节点，
可以传入一个参数，传入的参数是你要截止的兄弟节点

find():

查找元素里面的元素,对比$(‘div h2’), $(‘div’).find(‘h2’)的性能更高

children():

查找元素里面的所有子节点，括号里传入筛选的参数
$(‘ul’).children(‘li’); 只获取ul下面的li

和find()的差别是：

  1. find()能获取元素下的子节点，孙子节点无线往下延伸。
  2. children()只能获取子节点，子节点里面的节点无法获取。

eq():

alert($('#h2').index());  //当前元素在所有兄弟节点中的位置
})
</script>
</head>

</body>

toggleClass():

先判断元素有没有这个className,如果没有，就给给它添加，如果有就删除。

JQ中的宽高

width():

获取元素的宽度，但是没有单位 //alert($(‘div’).width());

赋值：$(‘div’).width(200);     //元素内容宽度变成200px
innerWidth(): 元素的宽度+padding
赋值：$(‘div’).innerWidth(200); //元素的总宽度是200，实际内容宽度需要减去padding值。

outerWidth():

元素的宽度+padding+border

赋值：$(‘div’).outerWidth(200); //元素的总宽度是200，实际内容宽度需要减去padding值和边框的宽度。

PS: 以上元素不带margin

outerWidth(true):

元素的宽度+padding+border+margin //()里面加true带margin

赋值：$(‘div’).outerWidth(200,true); //元素的占位宽度是200，实际内容宽度需要减去padding值和边框的宽度以及margin值。

与原生JS的区别：和原生offsetWidth一样，但是offsetWidth无法获取到隐藏元素的宽
outerWidth可以获取隐藏元素的宽

获取可视区的高：$(window).height(); //获取页面可视取的高度，宽度同理
获取页面的高：$(document).height(); //获取页面的高度，宽度同理

scrollTop():

鼠标的纵向滚动距离

scrollLeft():

鼠标的横向滚动距离
括号里可以传入一个数值，单位是PX，传入的数值会让滚动条滚动到这位置。
示例演示：

<script src="jquery-1.10.1.min.js"></script>
<script>
$(function(){
    $(document).click(function(){
        alert($(window).scrollTop());
    })
});
</script>
</head>
<body style="height:2000px">
    <div>div</div>
</body>

jQuery方法之DOM操作：

insertBefore():

把元素添加到指定元素的前面 

$('span').insertBefore('div');  //把span放到div的前面

before():

元素的前面位置应该是某元素 

$('div').before('span');        //在div的前面放入span

Ps: 这两个方法的行为是一样，但是后续的操作变了。

insertAfter():

把元素添加到指定元素的后面

$('span').insertAfter('div');//把span放到div的后面

after():

元素的后面是某元素 

$('div').after('span');  //在div的后面放入span

Ps: 这两个方法的行为是一样，但是后续的操作变了。

appendTo():

把元素添加到指定元素的里面的最后面 

$(‘span’).appendTo(‘div’);  //把span放到div的里面的最后位置

append():

元素的里面最后位置应该是某元素 

$(div).append(‘span’);    //div的里面的最后位置放入span

prependTo():

把元素放到指定元素的里面的开始位置 

$(‘span’).prependTo(‘div’);//把span放到div的里面的开始位置

prepend():

元素的里面的开始位置应该是某元素 

$(‘div’).prepend(‘span’);   //div的里面的开始位置放入span

示例演示：

<script src="jquery-1.10.1.min.js"></script>
<script>
$(function(){
    //$('span').insertBefore('div');  //把span放到div的前面
    //$('div').insertAfter('span');    //把div放到span的后面
    //$('div').appendTo('span');    //把div放到span里面的最后位置
    //$('div').prependTo('span');    //把div放到span里面的开始位置
    $('span').insertBefore('div').css('background','green');  //把span放到div的前面,然后span的背景变绿
    $('div').before($('span')).css('background','red');  //div的前面插入sapn，然后div的背景变红
})
</script>
</head>
<body>
    <div>div</div>
    <span>span</span>
</body>

remove():

删除某元素并且删除这个元素的所有操作行为 

$('div').remove();  //删除div元素

detach():

删除某元素但是保留这个元素的所有操作行为 

$('div').detach();  //删除div元素

示例：

<script src="jquery-1.10.1.min.js"></script>
<script>
    $(function(){
        $('#div1').click(function(){
            alert($('#div1').html());
        })
        var oDiv = $('#div1').detach();  //删除之后用变量存下
        $('body').append(oDiv);      //然后后面插入的时候还会保留它的事件操作
    });
</script>
</head>
<body">
    <div id="div1">div1</div>
</body>

复制某个元素。可以传入一个参数，参数为true，可以复制这个元素的操作行为

</script>
</head>
<body class="box">
    <div>div</div>

parent():

获取元素的父级 等于原生JS的parentNode
offsetParent(): 获取有定位的父级 等于原生JS的offsetParent一样。
示例：

<script src="jquery-1.10.1.min.js"></script>
<script>

    $('#div2').on('click',function(){
        $('#div2').parent().css('background','green'); //让div2的父级背景变成绿色
        $('#div2').offsetParent().css('background','green');  //让div2有定位的父级变成绿色，如果没有定位就默认body
    });
});
</script>
</head>
<body">
<div id="div1">
    <div id="div2"></div>
</div>
</body>

parents():

获取元素的所有祖先节点。 括号内可以传入需要筛选的元素

parentsUntil():

获取元素的所有祖先节点。 括号内可以传入需要到截止的元素

示例：

<script src="jquery-1.10.1.min.js"></script>
<script>
$(function(){
    $('#div2').parents().css('background','red');           //div2的所有父级背景换成红色
    $('#div2').parents('#div1').css('background','red');    //div2的父级只有div1变成红色
    $('#div2').parents('body').css('background','green');   //div2的父级只有body变成绿色
$('#div2').parentsUntil('#div').css('background','green');  //div2的父级到div1截止变成绿色

});
</script>
</head>
<body">
    <div id="div1">aaa
        <div id="div2">bbb</div>
    </div>
</body>

closest():

获取元素的最近的祖先节点。（包括元素自身，必须要写参数，并且只能找到一个元素）
示例：

<script src="jquery-1.10.1.min.js"></script>
<script>
$(function(){
    $('#div2').closest('.box').css('background','red');

                                //如果元素自身加了class="box"，那么只会找到它自身
</script>
</head>
<body class="box">
    <div id="div1" class="box">aaa
        <div id="div2">bbb</div>
    </div>
</body>

关于this ：

在循环中，this在原生JS中是不存在的，只有在事件中才能存在，但是在JQ的each()之中，this是存在的。This相当于括号里第二个参数，指的是元素本身。this等于li。

关于跳出循环：

原生之中，如果要跳出循环，使用的是break。JQ之中跳出循环就不能用break了，要用return false；

jQuery方法之事件操作：

事件操作注意: 在JQ中所有的事件操作都是绑定的形式

on():

执行某事件。
空号里两个参数，第一个表示事件，第二个表示事件执行的函数
可以同时填写多个事件，不用再分开写
并且还可以以JSON的形式，对不同事件执行不同的函数
示例演示：

<script src="jquery-1.10.1.min.js"></script>
<script>
$(function(){

    //不同事件执行同一函数
    $('div').on('click mouseover',function(){
        alert(123);
    })

    //不同事件执行不同函数                                            
    $('div').on({
        'click' : function(){ alert(123); },
        'mouseover' : function(){ alert(456); }
    })

})
</script>
</head>
<body>
    <div>div</div>
</body>

关闭某事件。可以全部关闭，也可以只关闭某一件事件

<script src="jquery-1.10.1.min.js"></script>

$(function(){
    $('div').on('click mouseover',function(){
        alert(123);
        $('div').off();  //关闭所有事件
        $('div').off('mouseover');  //只关闭mouseover事件

});
</script>
</head>
<body>
    <div>div</div>
</body>

delegate()：

事件绑定 冒泡机制 //$(‘ul’).delegate(‘li’,’click’,function(){});

undelegate()：

解除事件绑定 //针对的是dalegate(),不用能off()来解除绑定
示例：

<script src="jquery-3.2.0.js"></script>
    <script>
    $(function(){
        $('input').click(function(){
            $li = $('<li>hello</li>');       //手动创建li
            $('ul').append($li);
        })

        $('ul').delegate('li','click',function(ev){        
            $(this).css('background','red');  
            $('ul').undelegate();
        })
        //1.this在事件委托之中，指向的是委托的对象，这里指向的是li
        //2.如果要找到ul有两种方式：
                                1)  $(‘ul’);   
                                2)  ev下有一个对象，ev.delegateTarget 指向的就是ul $(ev.delegateTarget)  
        // $('li').click(function(){
        //           $(this).css('background','red');
        // })
        //
        // 通过以上代码可以看出事件绑定的一个优势。
        // 1.普通事件在后续添加的li面前是不能执行的，因为在后续添加的li之前，事件操作就已经加载结束了。
        // 2.事件绑定可以实现让后续添加的li也拥有事件行为。因为事件是绑定在父级ul身上，只要父级不变，里面元素都不会受影响。
    })

</script>
</head>
<body>

jQuery方法之事件细解：

ev:

等于原声event对象；

ev.pageX

(相对于整个页面的鼠标位置)

ev.clientX

(相对于可视区的) 鼠标位置

ev.pageY

(相对于整个页面的鼠标位置)

ev.clientY

(相对于可视区的) 鼠标位置

ev.which:

等于原声event.keyCode，键盘键值。
示例：

$(function(){
    $(document).keydown(function(ev){
        alert(ev.which)                     //弹出你按下键盘的键值
    })
})();

ev.preventDefault:

阻止默认事件

ev.stopPropagetion:

阻止冒泡事件

return false: 阻止冒泡事件 + 阻止默认事件

ev.data:

数据传参
示例：

<script src="jquery-1.10.1.min.js"></script>
<script>
$(function(){
    $('div').on('click',{name:'hello'},function(ev){
        alert(ev.data.name);              //传入json中name的值        //hello
        alert(ev.target);                 //找到当前操作的事件对象是谁  //div
        alert(ev.type);                      //找到当前操作的事件是什么类型的事件   //click
    })
});
</script>
</head>
<body>
    <div>123213</div>
</body>

获取到有定位的父级的距离.需要把当前元素转化为有定位的形式,因为它不认margin值

<script src="jquery-1.10.1.min.js"></script>

JQ的运动

animate()：

运动方法 可以传参数
第一个参数是个json，里面传入属性和属性值
第二个参数是时间，默认400毫秒，调节运动快慢
第三个参数是运动形式，有两种运动形式（默认swing(慢快慢)，linear(匀速)）
第四个餐宿是回调函数。

stop()：

停止运动
默认阻止当前运动，不会阻止后面的运动
括号里写true，就可以阻止后续所有运动。 元素.stop(true);
括号里如果写两个true，会立即停止到当前运动的目标点，并且停止后续运动
元素.stop(true,ture);

finish()：

停止运动 立即停止到你所有运动指定的目标点。 元素.finish();

delay()：

延迟运动，括号传入参数是延迟的时间
示例：

$(this).animate({width:300},600).delay(1000).animate({height:300},600);

综合示例：

<style>
    #div1 { width: 100px; height: 100px; background: red; }
    #div2 { width: 100px; height: 100px; background: red; margin-top: 10px; }
</style>
<script src="jquery-1.10.1.min.js"></script>
<script>
$(function(){
    $('#div1').click(function(){
        $(this).animate({width:300},600).delay(1000).animate({height:300},600);
    });
    $('#div2').click(function(){
        $('#div1').stop();  //默认阻止当前运动，不会阻止后面的运动
        $('#div1').stop(true);  // 停止后续所有运动
        $('#div1').stop(true,true);  // 立即停止到当前运动的目标点，并且停止后续运动
        $('#div1').finish();  //立即停止到所有运动指定的目标点
    })
});
</script>
</head>
<body>
    <form>
        <div id="div1"></div>
        <div id="div2"></div>
    </form>
</body>

$下的常用方法

$.type():

判断元素的类型
和typeof一样，但比他更强大，可判断的类型更细致更多

$.trim():

去除文本的前后空格
示例：

$(function(){
    var str = '   hello   ';
    alert('('+str+')');           //弹出str的内容是带空格的。
    alert('('+str.trim()+')');   //弹出str的内容是去除空格的。
});

$.inArray():

类似于indexOf，不过是返回数组的下标位置。
括号内传入两个值，第一个是你要查找的数组内容，第二个参数是数组的名称
示例：

$(function(){
    var arr = ['a','b','d','d'];
    alert($.inArray('b',arr));   //弹出b在数组的位置
});

$.proxy():

改变this指向
括号传入两个参数，第一个是函数的名称，第二个是需要指向谁的名称。
如果要调用函数，需要在后面加上();
示例：

$(function(){
    function show(a,b){
        alert(a+':'+b)
        alert(this);
    }
    //show();                          //这里的this指向是window
    //$.proxy(show,document)();        //这里的this指向是document。
    //$.proxy(show,document)(3,4);     //传参的参数可以写在后面的括号里
    //$.proxy(show,document,3,4)();    //参数也可以写在开始的括号里，第三位开始可以接受参数
    $.proxy(show,document,3)(4);       //参数还可以混着写
});

$.noConflict():

防止$冲突 ，可以用其他单词符号替换$的作用
示例：

var miaov = $.noConflict();  //用此方法可以直接用miaov代替$在JQ中的作用
miaov(function(){
    miaov('body').css('background','red');
})

$.parseJSON():

把字符串转变成JSON
这个字符串必须是严格模式下的JSON
示例：

var str = '{"name":"miaov"}'    //必须写成这样的严格模式。
alert($.parseJSON(str).name);  //这样就可以把字符串转成可以使用的JSON

$.makeArray():

把类数组变成真正的数组
类数组：一种有长度和下标的元素叫做类数组
示例：

var aDiv = document.getElementsByTagName('div'); //这种有长度和下标的元素叫做类数组
//aDiv.push();              //错误的写法，必须通过下面的方法转成数组
$.makeArray(aDiv).push();  //通过此方法可以变成真正的数组，不会报错

JQ的Ajax使用

$.ajax():

JQ下的ajax使用方法

$.ajax({
    url : 'xxx.php',                 //地址
    data : 'name=hello&age=20',      //数据
    type : 'POST',                    //数据传输方式
    success : function(data){        //成功之后执行的参数
        alert(1);
    },
    error : function(){                 //失败之后执行的参数
        alert(2);
    }
    contentType                     //请求头信息，post请求的时候需要写一下
    dataType                        //返回的数据类型
    cache                            //是否缓存
    Timeout                            //是否延迟
})

示例：

$.get(‘url?+你要的传的值’function(){});   //两种传数据的方式
$.get(‘url’,{name:’hello’},function(){});

JQ下的ajax的细分方法，post方式

示例：$.post(‘url’,function(){});
